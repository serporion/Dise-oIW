    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $min-size

    Tamaño mínimo del texto.

    		Number— none
    $max-size

    Tamaño máximo del texto.

    		Number— none
    $min-width

    Ancho mínimo del viewport (por defecto, 320px).

    		Number— none
    $max-width

    Ancho máximo del viewport (por defecto, 1200px).

    		Number— none

    Returns

    String —

    ensure-contrast

    @function ensure-contrast($color, $background, $minimum) { ... }

    Description

    Función para asegurar contraste entre un color y su fondo. Si el contraste no cumple el nivel mínimo especificado, ajusta el color. Usa la luminosidad relativa para el cálculo.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $color

    Color principal a evaluar.

    		Color— none
    $background

    Color de fondo (por defecto, blanco).

    		Color— none
    $minimum

    Nivel mínimo de contraste (por defecto, 4.5).

    		Number— none

    Returns

    Color —

    Devuelve el color ajustado si no cumple con el contraste.

    Used by

    • [mixin] navigation
